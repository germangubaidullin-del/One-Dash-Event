##  Pitch

**One Dash Event** — це не просто каталог коктейлів, а інтерактивна платформа, що поєднує барну культуру та сучасні веб‑технології.  
Мета проєкту — створити стильний, мінімалістичний та зручний сервіс, де користувачі можуть:
- знаходити коктейлі за інгредієнтами, подіями чи посудом,
- відкривати нові рецепти з фото, музичним супроводом та підказками,
- оцінювати напої та ділитися власними ідеями.

У майбутньому **One Dash Event** планується розвивати як комерційний продукт із реєстрацією користувачів, підтримкою кількох мов, адаптацією під різні пристрої та навіть десктопною програмою.

## Опис
Веб‑додаток на Flask для перегляду та пошуку коктейлів.  
Проєкт включає:
- Категорії коктейлів (вечірка, романтика, класика тощо).
- Пошук за назвою та інгредієнтами.
- Детальні сторінки коктейлів з фото, інгредієнтами та кроками приготування.
- Навігацію через меню та контекстну кнопку «назад».

Цей сайт створений як частина портфоліо Herman.

## Запуск проєкту

1. Клонувати репозиторій:
   ```bash
   git clone <repo-url>
   cd project-folder

2. Створити віртуальне середовище:
python -m venv venv
venv\Scripts\activate   # для Windows

3. Встановити залежності:
pip install -r requirements.txt

4.Запустити Flask:
flask run

5. Відкрити браузер та ввести http://127.0.0.1:5000/

13. Категорія «шоу та екстремальні коктейлі».

15. Реєстрація користувачів.
16. Можливість пропонувати власні рецепти.
17. Адаптація під різні пристрої (responsive design).

20. Комерційна складова (монетизація).
